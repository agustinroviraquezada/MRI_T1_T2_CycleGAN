## Introduction

MRI T1-T2 CycleGAN is a deep learning model that performs image conversion between T1-weighted (T1) and T2-weighted (T2) MRI images. Learn to convert MRI images from one modality to another modality without using training data pairs. This model uses a consistent cycle loss to bring the transformed image closer to the original input when returning to the source domain.Â
